# Email Sequence Design

Create automated email flows that nurture and convert.

## Sequence Types

**Welcome Sequence:**
- Email 1: Welcome + set expectations
- Email 2: Deliver value (guide, tips)
- Email 3: Social proof
- Email 4: Soft pitch

**Trial Sequence:**
- Email 1: Setup help (activation)
- Email 3: Feature highlight
- Email 5: Upgrade prompt
- Email 7: Expiring soon

**Nurture Sequence:**
- Educate → Build trust → Soft pitch

## Process

### Step 1: Define Goal
- Activate free users?
- Convert trials?
- Re-engage churned?

### Step 2: Map Journey
- Day 0 → Day 1 → Day 3 → Day 7
- What does each email achieve?

### Step 3: Write Emails
- Subject line (clear benefit)
- Body (value-first)
- CTA (single, clear)

### Step 4: Set Timing & Triggers
- Time-based (Day 3)
- Behavior-based (Clicked link)
- Condition-based (Hasn't upgraded)

## Output
Email sequence with copy, timing, and triggers.

## Quality Bar
- Each email has clear purpose
- Progressive disclosure (don't overwhelm)
- Personalized where possible
- Clear unsubscribe
