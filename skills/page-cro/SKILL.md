# Landing Page CRO Audit

Analyze and optimize pages for higher conversion rates.

## Audit Framework

### 1. Value Proposition Clarity (5-second test)
- Can a visitor understand what you offer in 5 seconds?
- Score: 1-5

### 2. Headline Effectiveness
- Specific (not generic)?
- Benefit-clear?
- Differentiated?
- Score: 1-5

### 3. CTA (Call-to-Action)
- Visible above fold?
- Action-oriented copy?
- Clear what happens next?
- Repeated throughout page?
- Score: 1-5

### 4. Visual Hierarchy
- Scannable (short paragraphs, bullets)?
- Clear flow (eye naturally moves down page)?
- Key elements stand out?
- Score: 1-5

### 5. Trust Signals
- Social proof (testimonials, logos, stats)?
- Security/privacy badges?
- Guarantees?
- Score: 1-5

### 6. Objection Handling
- Addresses common objections?
- FAQ section?
- Risk reversal (free trial, guarantee)?
- Score: 1-5

### 7. Friction Points
- Form too long?
- Unclear navigation?
- Slow load time?
- Mobile issues?

## Process

### Step 1: Audit Page Against Framework
Score each element 1-5, identify gaps.

### Step 2: Prioritize Recommendations
**Quick Wins:** High impact, low effort
**High-Impact Changes:** Worth A/B testing
**Long-Term:** Strategic improvements

### Step 3: Provide Copy Alternatives
Not just "improve headline" — provide 3 specific options.

## Output Format

```markdown

## Quick Assessment
- Value Prop Clarity: [Score] — [Note]
- Headline: [Score] — [Note]
- CTA: [Score] — [Note]
- Visual Hierarchy: [Score] — [Note]
- Trust Signals: [Score] — [Note]
- Objection Handling: [Score] — [Note]

## Quick Wins (Do Now)
1. **[Change]:** [Why + expected impact]
2. **[Change]:** [Why + expected impact]

## High-Impact Changes (A/B Test)
1. **[Change]:** [Why + effort estimate]

## Test Ideas
1. **Hypothesis:** If we [change], then [metric] will improve because [reason]
   - **Test:** [What to test]
   - **Measure:** [Primary metric]

## Copy Alternatives

### Current Headline
[Existing headline]

### Recommended Options
1. **[Option 1]** — [Rationale]
2. **[Option 2]** — [Rationale]
3. **[Option 3]** — [Rationale]

### Current CTA
[Existing CTA]

### Recommended Options

- [Option 2]
```

## Quality Bar
- Recommendations are specific (not "improve headline")
- Prioritized by impact
- Test ideas are proper hypotheses
- Provides copy alternatives
