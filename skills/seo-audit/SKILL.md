# SEO Audit

Audit website for technical, on-page, and content SEO issues.

## Audit Areas

### 1. Technical SEO
- Site speed (Core Web Vitals)
- Mobile-friendliness
- Crawlability (robots.txt, sitemap.xml)
- HTTPS
- Canonical URLs
- Redirect chains

### 2. On-Page SEO
- Title tags (unique, descriptive, keyword-targeted)
- Meta descriptions (compelling, under 160 chars)
- Header tags (H1-H6 hierarchy)
- Image alt text
- URL structure (descriptive, lowercase, hyphens)
- Internal linking

### 3. Content SEO
- Keyword targeting (clear intent)
- Content depth and quality
- Duplicate content
- Thin content (low-value pages)

### 4. Off-Page SEO
- Backlink profile
- Domain authority
- Toxic links
