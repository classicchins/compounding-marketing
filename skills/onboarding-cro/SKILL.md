# Onboarding & Activation Optimization

Optimize post-signup experience to get users to "aha moment" faster.

## Key Metrics
- **Activation rate:** % who reach aha moment
- **Time to value:** How long to first value
- **Feature adoption:** % using core features

## Process

### Step 1: Identify Aha Moment
- What action correlates with retention?
- When do users "get it"?

### Step 2: Map Current Journey
- Signup → [Step 1] → [Step 2] → Aha Moment
- Where do users drop off?

### Step 3: Reduce Steps
- Can you skip steps?
- Can you pre-populate data?
- Can you defer non-essential tasks?

### Step 4: Guide Users
- Checklists (visible progress)
- Tooltips (contextual help)
- Empty states (prompt action)
- Success messaging (celebrate wins)

## Output
Provide optimized onboarding flow with reduced friction and clear path to value.
