# Signup Flow Optimization

Optimize signup/registration flows for higher completion rates.

## Key Principles
1. **Minimize fields** (only ask what's essential)
2. **Clear labels** (no jargon)
3. **Show progress** (multi-step forms)
4. **Reduce anxiety** (privacy, security messaging)
5. **Mobile-first** (optimize for mobile)

## Audit Areas
- Number of fields (each field = drop-off)
- Field labels (clear?)
- Error messages (helpful?)
- Button copy (action-oriented?)
- Social signup (Google, GitHub, etc.)
- Progress indicator (multi-step)

## Output
Provide before/after flow, field recommendations, copy improvements.

## Quality Bar
- Each field justified
- Clear value proposition on signup page
- Mobile-optimized
