## The Compound Chain

Skills build on each other. Recommended order for new product launches:

```
1. cm-context → Product-marketing context file
2. icp-research → Ideal customer profile
3. competitive-analysis → Competitive landscape
4. positioning → Market position
5. messaging-framework → Message pillars
6. value-proposition → Value props per segment
7. copywriting → Homepage + pricing page
8. seo-audit → SEO foundation
9. launch-strategy → Launch plan
10. email-sequence → Activation emails
```

**Why this order matters:**

- Research informs positioning
- Positioning informs messaging
- Messaging informs copy
- Copy informs outreach
- Everything compounds

## How to Read Skill Output

Every skill outputs to `deliverables/[skill-name].md`.

**Typical structure:**

1. **Executive Summary** — TL;DR of findings/recommendations
2. **Framework Used** — What methodology guided the work
3. **Analysis/Process** — Step-by-step work
4. **Output/Deliverable** — The actual asset (copy, plan, doc)
5. **Next Steps** — What to do with this deliverable
6. **Related Skills** — What to run next

**Quality bar:**

- Specific (not generic)
- Actionable (clear next steps)
- Evidence-based (cites research, data, or framework)
- No AI slop (no "comprehensive", "seamless", "robust")
