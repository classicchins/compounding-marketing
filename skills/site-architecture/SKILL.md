# Site Architecture for SEO

Design URL structure and site hierarchy for optimal SEO and user experience.

## Principles

### 1. Flat Structure (3 clicks to any page)
```
Homepage → Category → Page
```
Not:
```
Homepage → Section → Subsection → Category → Subcategory → Page
```

### 2. Descriptive URLs
**Good:** `/features/real-time-collaboration`
**Bad:** `/page?id=12345`

### 3. Consistent Hierarchy
```
/features/
/features/real-time-collaboration
/features/version-history

/pricing/
/pricing/teams
/pricing/enterprise
```

### 4. Internal Linking
- Link from high-authority pages to important pages
- Use descriptive anchor text
- Create topical clusters (pillar + cluster pages)

## Process

### Step 1: Map Content Types
- Marketing pages (features, pricing, about)
- Blog/content
- Product docs
- Legal pages

### Step 2: Define URL Structure
```
/features/[feature-name]
/blog/[post-slug]
/customers/[customer-name]
/vs/[competitor-name]
```

### Step 3: Plan Navigation
- Primary nav (top 5-7 links)
- Footer nav (comprehensive)
- Breadcrumbs (for deep pages)

### Step 4: Create Internal Linking Strategy
- Pillar content links to cluster content
- Cluster content links back to pillar
- Related content links (end of blog posts)

## Output

Site architecture diagram + URL structure guidelines.

## Quality Bar
- Flat structure (3-click rule)
- Descriptive, consistent URLs
- Clear navigation hierarchy
- Strategic internal linking
