# /cm:standup — Marketing Standup

Structured marketing standup for async teams or solo accountability.

## What It Does

A focused 5-minute standup format adapted for marketing work. Perfect for async team updates, accountability partners, or self-management.

## Process

1. **Yesterday's Marketing**
   - What marketing work shipped yesterday?
   - What moved from "in progress" to "done"?
   - Any experiments completed?

2. **Today's Marketing**
   - What marketing work starts or continues today?
   - What's the deliverable?
   - When will it be done?

3. **Blockers**
   - What's slowing you down?
   - Waiting on: [assets, approvals, data, decisions]?
   - Need help with: [copywriting, design, technical, strategic]?

4. **Metrics Check** (optional)
   - Any metric that changed significantly?
   - Anything worth noting for the team?

## Output Format

```markdown
# Marketing Standup — [Date]

## Yesterday
- ✅ [Completed task 1]
- ✅ [Completed task 2]
- 🔄 [Continued work on X]

## Today
- 🎯 [Priority 1] — [ETA/deliverable]
- 📝 [Priority 2] — [ETA/deliverable]
- 💡 [Priority 3] — [ETA/deliverable]

## Blockers
- ⚠️ [Blocker 1] — Waiting on [person/thing]
- ⚠️ [Blocker 2] — Need help with [specific ask]

## Quick Metrics
- [Metric]: [value] ([trend])
```

## When to Use

- Daily team syncs (async or live)
- Solo accountability (morning routine)
- Before focus blocks (clarify what you're doing)

## Time Investment

5 minutes

## Output

- Clear accountability on yesterday
- Clear commitment for today
- Visible blockers (easier to get help)

## Best Practices

1. **Be specific** — "Write homepage copy" not "work on website"
2. **Include deliverables** — "Draft ready for review by 3pm"
3. **Surface blockers early** — Don't wait until standup to ask for help
4. **Keep it short** — This isn't a status report, it's a pulse check

## Philosophy

The standup is a forcing function for clarity. If you can't articulate what you did, what you'll do, and what's blocking you — you don't have clarity. Fix that before starting work.
