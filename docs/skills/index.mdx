## How to Use Skills

1. **Set up context** — Create your `.cm-context` file
2. **Run the skill** — Type the skill command (e.g., `/cm:position`)
3. **Review output** — The AI generates a deliverable
4. **Iterate** — Refine based on your feedback

## Skill Depth

Some skills have parameters for depth:

```
/cm:position --depth=full
```

- `--depth=quick` — Fast output, minimal research
- `--depth=standard` — Balanced (default)
- `--depth=full` — Comprehensive, includes research

Explore the individual skill pages for detailed usage.
