# Ad Creative & Copywriting

Write high-performing ad copy for paid campaigns.

## Platform Best Practices

**Google Search Ads:**
- Headline: Include keyword + benefit
- Description: CTA + differentiator
- Extensions: Sitelinks, callouts

**Facebook/Instagram Ads:**
- Hook: First line grabs attention
- Visual: High-quality image/video
- CTA: Clear button ("Learn More", "Sign Up")

**LinkedIn Ads:**
- Professional tone
- B2B value prop
- Lead gen forms (reduce friction)

## Ad Copywriting Framework

### Headline
- Include keyword (Google)
- Promise benefit
- Create curiosity

### Body
- Problem + solution
- Social proof (if space)
- Clear CTA

### Visual
- Product in action
- Before/after
- Customer using product

## Process

### Step 1: Research
- What competitors are running? (Facebook Ad Library)
- What messaging resonates?

### Step 2: Write Variations (3-5)
- Test different hooks
- Test different CTAs

### Step 3: Design Creative
- Match ad copy
- Platform specs (size, format)

## Output
Ad copy variations + creative concepts.
