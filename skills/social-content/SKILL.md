# Social Media Content Creation

Create platform-specific social content that drives engagement and aligns with marketing goals.

## Process

### Step 1: Define Goal
- **Awareness:** Reach new audience
- **Engagement:** Start conversations
- **Traffic:** Drive to content/product
- **Community:** Build relationships

### Step 2: Choose Platform & Format

**Twitter/X:**
- Threads (educational, storytelling)
- Single tweets (insights, hot takes)
- Engagement bait (questions, polls)

**LinkedIn:**
- Long-form posts (insights, lessons)
- Carousels (frameworks, guides)
- Case studies / results

**Best practices per platform:**
- Twitter: Short, punchy, conversational
- LinkedIn: Professional, insight-driven, longer OK

### Step 3: Create Content

**Thread structure:**
1. Hook (strong opening)
2. Context (set up the story/lesson)
3. Value (actionable insights)
4. CTA (what to do next)

**Single post structure:**
- Lead with insight
- Support with example or data
- End with takeaway

### Step 4: Apply Brand Voice
- Match tone from brand-voice skill
- Use vocabulary guidelines
- Maintain consistency

## Output

Provide formatted content:

```markdown
## [Platform] [Format]: [Topic]

**Goal:** [Awareness/Engagement/Traffic]

**Content:**

[Full post or thread, formatted for platform]

**Notes:**
- [Any timing, hashtag, or engagement tips]
```

## Quality Bar
- Platform-appropriate format and length
- Clear value or insight
- Matches brand voice
- Includes CTA when relevant
- Engaging hook
