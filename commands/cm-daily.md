# /cm:daily — Daily Marketing Review

Quick daily check-in on marketing performance and priorities.

## What It Does

A 10-minute daily review that keeps you informed on what's live, what's working, and what needs attention — without getting lost in dashboards.

## Process

1. **What's Live Today**
   - Ask: What campaigns, content, or experiments are currently active?
   - Check: Any new content published in last 24h?
   - Check: Any ads running?
   - Check: Any email sequences active?

2. **Performance Snapshot**
   - Ask: What metrics moved yesterday? (Traffic, signups, trials, MRR)
   - Ask: Any notable wins? (Viral post, PR mention, ranking improvement)
   - Ask: Any red flags? (Ad spend issues, conversion drops, negative feedback)

3. **Attention Required**
   - What needs action today?
   - Any time-sensitive opportunities?
   - Any fires to put out?

4. **Today's Marketing Focus**
   - Ask: What's the ONE most important marketing task today?
   - Ask: What would make today a marketing win?

## Output Format

```markdown
# Daily Marketing Review — [Date]

## What's Live
- **Campaigns:** [Active campaigns/ads]
- **Content:** [Recently published]
- **Sequences:** [Active email flows]

## Performance Snapshot
- **Traffic:** [trend vs yesterday/last week]
- **Signups:** [count + trend]
- **Key metric:** [whatever matters most]

## Wins
- [Notable positive signal]

## Flags
- [Anything concerning]

## Today's Focus
1. **Priority:** [The ONE thing]
2. **Quick wins:** [Optional 2-3 small tasks]

## Notes
- [Anything to remember]
```

## When to Use

- Start of each workday (before diving into tasks)
- After a launch (daily during launch week)
- When managing multiple campaigns

## Time Investment

10-15 minutes

## Output

- Quick mental model of marketing state
- Clear priority for the day
- Early warning on problems

## Philosophy

Daily awareness prevents surprises. The goal isn't deep analysis — it's staying oriented. Save deep dives for `/cm:weekly`.
